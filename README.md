# Cybersecurity-Internship
Developershub Corporation Cybersecurity Inernship
🔐 Cybersecurity Internship Project
📌 Overview

This project demonstrates implementation of security best practices in a Node.js authentication system.

🛡 Security Features Implemented

Input Validation using validator

Password Hashing using bcrypt

Token-Based Authentication using JWT

HTTP Header Security using Helmet.js

🧪 API Endpoints
🔹 POST /signup

Registers new user
Returns JWT token

🔹 POST /login

Authenticates user
Returns JWT token

🔧 Technologies Used

Node.js

Express.js

bcrypt

jsonwebtoken

Helmet.js

Postman (for testing)
